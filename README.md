# Smart_market

Projeto em Python para registrar compras de mercado, comparar preços entre períodos e ajudar no controle de gastos.

## Objetivos
- Registrar compras manualmente
- Comparar preços entre meses
- Calcular totais automaticamente
- Futuramente ler notas fiscais por imagem

- models
- services
- utils
- data
- interface
