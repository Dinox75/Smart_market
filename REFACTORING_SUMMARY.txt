🔌 INTEGRAÇÃO API
═══════════════════════════════════════════════════════════════════════════════

BaseURL: http://localhost:5000/api

ENDPOINTS UTILIZADOS:
  GET /api/compras
    └─ Retorna: { success, compras[] }
    └─ Uso: Buscar todas as compras

  POST /api/compras
    ├─ Body: { mercado, data, hora, itens[] }
    └─ Retorna: { success, compra, message }
    └─ Uso: Registrar nova compra

  GET /api/dashboard (futuro)
    └─ Retorna: { success, dashboard { metrics } }
    └─ Uso: Dados agregados do dashboard

TRATAMENTO DE ERROS:
  ├─ Try/Catch em todas as requisições
  ├─ Toast notifications
  ├─ Console logging para debug
  └─ Fallback para estado vazio

🚀 COMO USAR
═══════════════════════════════════════════════════════════════════════════════

1. ATIVAR AMBIENTE VIRTUAL:
   └─ .\venv\Scripts\Activate.ps1

2. INSTALAR DEPENDÊNCIAS:
   └─ pip install -r requirements.txt

3. INICIAR SERVIDOR:
   └─ python app.py
   └─ 🟢 Flask rodando em http://localhost:5000

4. ABRIR NO NAVEGADOR:
   └─ http://localhost:5000
   └─ 🎨 Interface carrega com sucesso

5. USAR A APLICAÇÃO:
   └─ Dashboard: Visualizar dados
   └─ Registrar: Adicionar nova compra
   └─ Histórico: Ver compras anteriores
   └─ Comparar: Análise entre compras
   └─ Análise: Top categorias
   └─ Sobre: Informações do projeto
