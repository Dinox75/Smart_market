Smart Market

Controle inteligente e profissional das suas compras de supermercado com sincronização em tempo real

O Projeto

Smart Market é uma aplicação web moderna desenvolvida para ajudar você a gerenciar suas compras de supermercado de forma inteligente e eficiente. A plataforma oferece ferramentas poderosas para registrar, analisar e comparar compras, ajudando você a entender seus padrões de gasto.

Funcionalidades Principais

Registrar Compras

Registre todas as suas compras com detalhes de mercado, data, hora e itens

Dashboard Interativo

Visualize estatísticas em tempo real sobre seus gastos

Histórico Completo

Acesse todo o histórico de compras e navegue pelos detalhes

Comparar Compras

Compare duas compras para entender variações de preço

Análise de Categorias

Identifique em quais categorias você gasta mais

Sincronização em Tempo Real

Dados atualizados instantaneamente quando mudanças são feitas

Tecnologias Utilizadas

FrontendHTML5, CSS3, JS
BackendFlask, Python 3
Banco de DadosJSON local
GráficosChart.js
ÍconesFont Awesome 6
APIREST com CORS

Fluxo da Aplicação

1. Registrar Compra

Você acessa a seção "Registrar" e preenche os detalhes: mercado, data, hora e itens

2. Enviar para Servidor

Os dados são enviados via API REST para o servidor Flask, que valida e categoriza os produtos

3. Armazenamento

A compra é salva em um arquivo JSON no servidor

4. Sincronização em Tempo Real

O frontend busca automaticamente os dados atualizados a cada 5 segundos

5. Visualização de Dados

Você pode visualizar o dashboard, histórico e análises com gráficos atualizados

Como Usar

Começando

  1. Clique em "Registrar" para adicionar sua primeira compra
  2. Preencha o mercado, data e hora da compra
  3. Adicione os itens com produto, preço unitário e quantidade
  4. A categoria é detectada automaticamente
  5. Clique em "Salvar Compra"
  6. Visualize seus dados no Dashboard
